⚡ БЫСТРЫЙ ЗАПУСК (краткая версия):

1. Установить PostgreSQL, Python 3.11+, Node.js 18+, Ollama
2. Создать БД: legal_cms
3. Backend:
   cd backend
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   alembic upgrade head
   python create_admin.py
   uvicorn app.main:app --reload

4. Frontend:
   cd frontend
   npm install
   npm run dev

5. Ollama:
   ollama serve
   ollama pull deepseek-ocr:latest

6. Открыть: http://localhost:5173
   Логин: admin
   Пароль: admin123
